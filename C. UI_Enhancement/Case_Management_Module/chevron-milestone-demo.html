First segment (straight left edge, no notch):
polygon(0 0, calc(100% - var(--tip)) 0, 100% 50%, calc(100% - var(--tip)) 100%, 0 100%)

All other segments (arrow right + V-notch left):
polygon(0 0, calc(100% - var(--tip)) 0, 100% 50%, calc(100% - var(--tip)) 100%, 0 100%, var(--tip) 50%)

The 6th point var(--tip) 50% is the V-notch tip. Move it to change notch depth:
ChangeEffect
--tip: 14pxShallower notch, gentler arrow (less overlap)
--tip: 20pxDefault — balanced shape
--tip: 28pxDeeper notch, sharper arrow (more overlap)

Overlap formula: margin-left: calc(var(--tip) * -1 + 2px)
The +2px keeps a hairline of the previous color visible so adjacent same-color segments don't merge into one blob.
